# CMF Project

Install with [Composer](https://getcomposer.org)

```sh
composer create-project orisai/cmf-project
```

Or via git

```sh
git clone git@github.com:orisai/cmf-project.git &&
cd cmf-project &&
rm -rf .git &&
git init &&
composer install
```
